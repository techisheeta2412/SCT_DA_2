# Data Cleaning and Preparation

## SkillCraft Technology - Data Analytics Task 2

## Project Overview

This project focuses on cleaning and preparing the Superstore dataset using Python and the Pandas library.

The objective of this task is to identify missing values, remove duplicate records, convert data types, and export a cleaned dataset for further data analysis and visualization.

## Dataset

The Superstore sales dataset contains information related to:

- Orders
- Customers
- Products
- Categories and Sub-Categories
- Geographic Regions
- Sales

## Data Cleaning Steps

1. Imported the dataset using Pandas.
2. Inspected the dataset using `head()` and `info()`.
3. Identified missing values using `isnull().sum()`.
4. Handled missing values in the Postal Code column.
5. Checked and removed duplicate rows.
6. Converted Order Date and Ship Date columns to datetime format.
7. Verified the cleaned dataset and data types.
8. Exported the cleaned dataset to a new CSV file.

The dataset was successfully cleaned and prepared for further data analysis and visualization. Missing values were handled, duplicate records were checked, date columns were converted to datetime format, and the cleaned dataset was exported as a CSV file.
